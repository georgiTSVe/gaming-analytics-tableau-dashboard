Gaming Analytics Dashboard (Tableau)

## Overview

This repository contains a Tableau dashboard created as part of a technical interview. The project analyzes player activity, gaming behavior, and payment patterns using synthetic test data to demonstrate data visualization, analytical thinking, and dashboard design skills.

The goal of the dashboard is to provide clear, actionable insights into revenue drivers, player engagement, and game performance.

---

## Business Questions Answered

* How much revenue is generated over time?
* Which games perform best in terms of player activity and payments?
* What are the behavioral patterns of active players?
* How do payment trends relate to gaming activity?

---

## Data Model

The dashboard is built on a structured analytical model consisting of fact and dimension tables:

**Fact Tables**

* **Gaming Data** – player gaming activity and usage metrics
* **Payments Data** – transaction-level payment information

**Dimension Tables**

* **Players** – player attributes
* **Games** – game metadata and categorization

This structure enables flexible slicing across players, games, and financial metrics.

---

## Key Metrics

* Total Revenue
* Number of Active Players
* Payments per Player
* Game-Level Performance
* Trend Analysis Over Time

## File Information

* **Format:** `.twbx` (Tableau Packaged Workbook)
* Contains all required data sources. No external connections are needed.

**Recommended Version:** Tableau Desktop 2025.x or newer.

---

## How to Open

1. Download the `.twbx` file from this repository.
2. Open it using Tableau Desktop.
3. Navigate through the dashboards to explore the insights.
